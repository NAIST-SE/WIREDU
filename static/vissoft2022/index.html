Demonstration of a GitHub Fork Visualization

This page provides visualization results in our paper:

Daigo Imamura, Takashi Ishio, Raula Gaikovina Kula, Matsumoto Kenichi.
Bug-Fix Variants: Visualizing Unique Source Code Changes across GitHub Forks.
Proceedings of VISSOFT 2022. NIER Track.

The preprint is available on arXiv.

Existing tools for investigating forks:

  • GitHub List of Forks: Case 1 (Obfuscator-LLVM) Case 2 (Clipy)
  • GitHub Network Graph: Case 1 (Obfuscator-LLVM) Case 2 (Clipy)

Our visualization shows unique commits in each fork.

It may takes around 10 seconds to download and visualize Case 1 depending on the environment.

Our view showing the top-10 active forks, as our prototype includes all commit messages in the datasets.
